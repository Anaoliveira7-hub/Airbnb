# Decifrando Lisboa: O que os Dados Revelam sobre o Alojamento Local na Capital 

## Objetivo do projeto

Explorar a dinâmica e os padrões do mercado de hospedagem em Lisboa. O estudo é destinado a analistas de dados (para analisar tendências), turistas planejadores (na busca por ofertas reais) e anfitriões estratégicos (para ajustarem preços facilmente).

## Ferramentas utilizadas
Python / Pandas
Matplotlib / Seaborn / Plotly
Scikit-Learn (Machine Learning)

## Resultado
A análise permitiu mapear os anúncios, identificando uma hiperconcentração no núcleo turístico e histórico, liderada de forma isolada por bairros como Santa Maria Maior e Misericórdia. Constatou-se que a distribuição do alojamento obedece a uma lógica de Pareto (80/20) na cidade. Descobriu-se que o "coração" do mercado opera com um preço padrão entre 80€ e 180€ , isolando-se os valores acima de 300€ como outliers para focar no alojamento típico. O estudo também expôs o perfil dos anfitriões, evidenciando o contraste entre a base de locais com 1 a 12 anúncios e o topo com apenas 4 mega-investidores monopolizando incríveis 1.088 imóveis

## Aprendizado
Limpeza e tratamento de dados;
Identificação de outliers de preços para delimitar o limite máximo considerado normal;
Análise de correlação geográfica e impacto socioeconômico (pressão habitacional);
Aplicação prática de clusters para desvendar monopólios no mercado imobiliário e classificar usuários de 0 a 3.
